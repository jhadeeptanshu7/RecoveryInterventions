Upload Data and Run a Pre-Trained Classifier for a Single Reddit User (Post based analysis)

  • Download example data set for using a pre-trained text based (Reddit post or tweet) classifier for a single user.
  • Click on the drop-down menu “Upload Data and Run a Pre-trained Classifier”.
  • Select “Single Reddit User”.
  • Select “Post based analysis”.
  • Click the "Browse/drop files here" space under “Upload user data zip file” and select the zipped folder previously downloaded. Alternatively, drag and drop the zipped folder from your file explorer onto the space.
  • Click “Submit”.
  • After the computation is complete you will be provided a link : “Click here to view the results.”
  • Click on the link. You will be directed to your results page.

Upload Data and Run a Pre-Trained Classifier for a Batch of Reddit Users (Post based analysis)

  • Download example data set for using a pre-trained text based (Reddit post or tweet) classifier for a batch of users.
  • Click on the drop-down menu “Upload Data and Run a Pre-trained Classifier”.
  • Select “Batch of Reddit Users”.
  • Select “Post based analysis”.
  • Click the "Browse/drop files here" space under “Upload user data zip file” and select the zipped folder previously downloaded. Alternatively, drag and drop the zipped folder from your file explorer onto the space.
  • Click “Submit”.
  • You will be provided with a Project Id. Make a note of it.
  • Click on the “Download Results” button.
  • Enter the Project id in the “Download Results” bar.
  • Click Submit.
  • If your results have been generated it will be downloaded automatically on your computer.
  • If your results have not been generated, a “Results for Project Id have not been generated yet” message will be displayed.
